# Getting Started with Claude Whisperer

The fastest way into Claude Whisperer is through the browser extension.

This repo has older Python research layers, but if your goal is to understand what Claude Whisperer is today, start with the extension and the refreshed top-level docs.

## Recommended Path

1. Clone the repository:

```bash
git clone https://github.com/ghostintheprompt/claude_whisperer.git
cd claude_whisperer
```

2. Install the extension from the `extension/` directory.

3. Open `claude.ai`.

4. Launch the extension and explore the current evaluation surfaces.

5. Export or review local results as needed.

## About the Older Python Layers

Directories like `core/`, `tools/`, and parts of `research/` come from earlier safeguard-monitor and comparative-testing phases of the project.

They are still worth reading if you want the repo's history, but they are not the quickest or cleanest way to understand the current surface.

## If You Are Here for Research Context

After you understand the extension, look at:
- `patterns/`
- `research/`
- selected docs in `docs/`

That order will give you the project logic without forcing you to start from stale framework assumptions.
