# Claude Whisperer Quickstart

If you want the repo running quickly, do this:

```bash
git clone https://github.com/ghostintheprompt/claude_whisperer.git
cd claude_whisperer
```

Then load the `extension/` folder as an unpacked browser extension.

After that:

1. Open `https://claude.ai`
2. Launch the extension
3. Explore the current evaluation surfaces
4. Generate a result
5. Export or review the local history if needed

## What This Quickstart Is For

- seeing the current repo surface quickly
- understanding the extension workflow
- confirming the project is alive

## What It Is Not For

- explaining every historical folder in the repo
- pretending the Python-side research layers are the main entry path
- promising a single neat framework where the repo itself is clearly mixed-era
